# HookLearn 案例整理总览

更新时间：2026-04-29

## 1. 项目定位

这个项目是一个以 Frida 为核心的游戏逆向与 Hook 学习仓库，主要覆盖：

- Android 设备连接与 Frida server 使用
- 进程枚举、Attach、Spawn 两种注入方式
- Unity Il2Cpp 信息采集、类方法定位、调用追踪
- Cocos 与 native so 层函数拦截
- 导出方法地址并生成 IDA 重命名脚本

## 2. 推荐学习顺序

1. 先完成环境准备与设备连接
2. 再练习进程查看与附加
3. 进入 Spawn 模式与通用 Hook 技巧
4. 学习 bridge 方式做 Unity/Il2Cpp 深入
5. 最后做导出与重命名，形成可复用分析流程

## 5. 快速跑通一条案例链

1. 按 doc/1.Frida-server使用.md 完成设备端准备
2. 用 doc/2.查看所有的进程.md 或 src/py/ListProcess.py 找到目标进程
3. 用 doc/4.Spawn模式的使用.md 或 src/AppRun.py 注入脚本
4. 进入 doc/7.Frida-brige.md 与 src/brige 目录做类/方法/地址采集
5. 按 doc/附导出json.md 产出 JSON 与 IDA 重命名脚本
