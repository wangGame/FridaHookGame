# HookLearn 案例打卡状态表

更新时间：2026-04-29

## 使用方法

- 每跑通一个案例，把状态从 `未开始` 改为 `进行中` 或 `已完成`
- 遇到问题时，在“问题记录”列写最短结论
- 建议每次只推进 1 到 2 个案例，保证可复现

状态说明：

- `未开始`：还没执行
- `进行中`：正在调试
- `已完成`：可重复跑通
- `阻塞`：当前无法推进
