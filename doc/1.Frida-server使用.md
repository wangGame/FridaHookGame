# 笔记

1.查看架构

```shell
adb shell getprop ro.product.cpu.abi
输出
x86_64
```

2.将文件传向android
```shell
adb push frida-server /data/local/tmp/
```

3.给与执行权限，启动

```shell
chmod 777 frida-server
```

4.attach system-server

```shell
frida -U -n system_server
```

system_server 是什么？

```
这是 Android 最核心进程：

负责：

ActivityManagerService

WindowManagerService

PackageManager

权限系统

前台应用管理

进程调度
```
